CSCI572 Assignment 3

In this assignment we will take our Apache Solr index constructed from Polar data that is now part of the Text Retrieval Conference and its Dynamic Domain track and that was originally crawled from the NSF ACADIS, NASA AMD, and NSIDC ADE sites and build a visual and interactive search interface for the data using the Data-Driven Documents framework


Banana

The Banana project was forked from Kibana, and works with all kinds of time series (and non-time series) data stored in Apache Solr. It uses Kibana's powerful dashboard configuration capabilities, ports key panels to work with Solr.

  • a. How about the distribution of sea ice? Which one have the more topics over sea ice?
  • b. Climate is regarded as the major driver of the disappearance of sea ice, which region concerns the climate problem most?
  • c. Is climate the only concern for that region? What's the 3 most popular topic over that region?
  • d. Also, what people talks about most in the past 10 years? And is it possible to predict the hottest topic in the next decade?

D3

D3.js is a JavaScript library for manipulating documents based on data. D3 helps you bring data to life. D3’s emphasis on web standards gives you the full capabilities of modern browsers without tying yourself to a proprietary framework.
